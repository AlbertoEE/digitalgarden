# Efecto
La comida que equipes en el zurrón antes del combate puede usarse en combate **sin coste de acción** (no consume ninguno de los 3 puntos de acción del jugador).

| Nivel | Efecto                                   |
| ----  | ---------------------------------------- |
| 1     | 1 ranura: usa 1 comida equipada gratis   |
| 2     | 2 ranuras: usa 2 comidas equipadas gratis |
| 3     | 3 ranuras: usa 3 comidas equipadas gratis |
