# Categorías de dado

En Sonos los dados usados son los siguientes. El dado más grande es de **mayor categoría**, el más pequeño de **menor categoría**. Estas categorías importan para habilidades especiales e interacciones.

| Dado | Categoría |
|------|-----------|
| 1d2  | 1         |
| 1d4  | 2         |
| 1d6  | 3         |
| 1d8  | 4         |
| 1d10 | 5         |
| 1d12 | 6         |
| 1d20 | 7         |
| 1d100| 8         |
