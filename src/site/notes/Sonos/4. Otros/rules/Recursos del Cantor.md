# Recursos de un Cantor

Un Cantor tiene varios recursos que puede mejorar de forma **temporal** o **permanente**:

- **Temporal**: [[Sonos/4. Otros/rules/rudimentos\|rudimentos]], [[Sonos/4. Otros/rules/Fundamentos\|Fundamentos]], Partituras; duración limitada o se pierden al usarse.
- **Permanente**: equipamiento. El avance puede depender del atributo más fuerte del [[Sonos/4. Otros/rules/instrumento vital\|instrumento vital]] (ej.: +2 de Vitalidad = +20 para Ritmo, +10 para Armonía).

Si los atributos del IV cambian **de forma permanente**, hay que recalcular recursos; si cambian **solo durante un combate**, no.

## Recursos

- [[Sonos/4. Otros/rules/Vitalidad\|Vitalidad]] — Lo que permite seguir en pie; a 0, forma animal → forma original con 1 PV, o inconsciencia/muerte.
- [[Sonos/4. Otros/rules/Silencio\|Silencio]] — Reducción de daño; dado por atributo del IV; pifia = daño verdadero, crítico = bloqueo total.
- [[Sonos/4. Otros/rules/Resonancia\|Resonancia]] — Recurso para ejecutar [[Sonos/4. Otros/rules/rudimentos\|rudimentos]]; máximo por atributo; se recupera con ejecuciones, Silencio perfecto, Pulso.
- [[Sonos/4. Otros/rules/Auralium\|Auralium]] — Catalizador del IV; hasta 3 usos para regenerar la mitad de la Vitalidad; recarga con descanso y Nana de la Súplica.
