# Descripción
Dulce pasta condensada hecha a partir de [[Sonos/4. Otros/items/Salvia de Smrug\|Salvia de Smrug]] que proporciona energía prolongada a quien lo ingiere.

# Efecto
Durante todo un combate consigue un extra de energía, que eleva tu velocidad base 3 casillas.
