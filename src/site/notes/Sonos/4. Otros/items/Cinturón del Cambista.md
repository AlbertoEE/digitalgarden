# Descripción
Cinturón de rudimentos de fabricación excepcional, con fundas de acceso rápido. Permite cambiar entre rudimentos equipados y guardados con una fluidez que roza lo sobrenatural.

# Efecto
Igual que [[Sonos/4. Otros/items/Cinturón de Rudimentos\|Cinturón de Rudimentos]] nivel 3: hasta 3 rudimentos guardados, cada intercambio con el [[Sonos/4. Otros/rules/instrumento vital\|instrumento vital]] cuesta 1 acción. **Objeto especial:** **2 veces por combate** puedes realizar un intercambio completo (equipado ↔ guardado en el cinturón) **sin coste de acción**.
