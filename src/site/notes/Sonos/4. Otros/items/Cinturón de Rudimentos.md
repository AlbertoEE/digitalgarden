# Efecto
Permite almacenar rudimentos para llevarlos al combate. Intercambiar un rudimento equipado en el [[Sonos/4. Otros/rules/instrumento vital\|instrumento vital]] por uno guardado en el cinturón **cuesta 1 acción** (solo sacar; meter ya está preparado) en lugar de 2.

| Nivel | Efecto                                              |
| ----  | ---------------------------------------------------- |
| 1     | 1 funda: intercambia hasta 1 rudimento por 1 acción  |
| 2     | 2 fundas: intercambia hasta 2 rudimentos por 1 acción cada uno |
| 3     | 3 fundas: intercambia hasta 3 rudimentos por 1 acción cada uno |
