# Descripción
Cristales enlazados cuanticamente capaces de eviar mensajes cortos a cualquier distancia entre ellos, uno actúa de oyente el otro de mensajero. Cada ciclo de recarga se establece el rol dependiendo de quien manda el mensaje primero.

# Materiales
- 2 x [[TTRPGS/campañas/Sonos/sistema/rules/Auralita Refinada\|Auralita Refinada]] 

# Preparación
Tiempo de preparación: 1 hora

| Tier    | Resultado                                           |
| ------- | --------------------------------------------------- |
| Pifia   | No ocurre, nada, pierdes los materiales             |
| 1       | Mensaje de 4 palabras, recarga 2 horas              |
| 2       | Mensaje de 7 palabras, recarga 60 min               |
| 3       | Mensaje de 10 palabras, recarga 30 min              |
| Crítico | Mensaje bidireciconal 10 palabras, recarga 10 min   |
