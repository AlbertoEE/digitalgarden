# Descripción
Líquido que contiene la esencia de [[TTRPGS/campañas/Sonos/lore/lugares/Klaos\|Klaos]] o [[TTRPGS/campañas/Sonos/lore/lugares/Veneas\|Veneas]] las lunas de Sonos. Se puede conseguir embotellando el agua de un lago dónde se reflejen dichas lunas.
