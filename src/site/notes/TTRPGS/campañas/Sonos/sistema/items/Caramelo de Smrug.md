# Descripción
Dulce caramelo hecho de [[TTRPGS/campañas/Sonos/sistema/items/Salvia de Smrug\|Salvia de Smrug]] que proporciona energía limitada a quien lo ingiere.

# Efecto
Durante 2 rondas consigue un extra de energía, tu movimiento sube X casillas dependiendo el nivel del [[TTRPGS/campañas/Sonos/sistema/items/Caramelo de Smrug\|Caramelo de Smrug]]

| Nivel | Efecto                           |
| ----  | -------------------------------- |
|  1    | Aumenta tu movimiento 2 casillas |
|  2    | Aumenta tu movimiento 3 casillas |
|  3    | Aumenta tu movimiento 5 casillas |
