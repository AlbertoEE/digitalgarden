# Descripción
Un frasco con un aroma embelesador que cambia la [[TTRPGS/campañas/Sonos/sistema/rules/Actitud\|Actitud]] de un NPC haciéndole más amigable. 

# Efecto
Este frasco a diferencia del [[TTRPGS/campañas/Sonos/sistema/items/Frasco de Fragancia Reconfortante\|Frasco de Fragancia Reconfortante]] puede usarse con un enemigo con el que ya se está combatiendo.

Un enemigo con el que ya se está combatiendo se volverá [[TTRPGS/campañas/Sonos/sistema/rules/Actitud\|Sospechoso]], olvidará cualquier agresión que haya sido inminente y responderá a cualquier otra agresión volviéndose hostil de nuevo.
