# Descripción
Cinturón de fundas diseñado para almacenar [[TTRPGS/campañas/Sonos/sistema/rules/rudimentos\|rudimentos]] y llevarlos al combate. Intercambiar un rudimento del [[TTRPGS/campañas/Sonos/sistema/rules/instrumento vital\|instrumento vital]] por uno guardado en el cinturón normalmente cuesta 2 acciones (sacar y meter). Con este cinturón, meter el rudimento ya está preparado; solo pagas la acción de sacar el que sustituyes.
{ #cinturon-rudimentos}


# Materiales
- Cuero tratado

# Preparación
Tiempo de preparación: 8 horas
